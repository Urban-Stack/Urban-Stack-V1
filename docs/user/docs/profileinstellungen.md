# Profileinstellungen

Die [Profileinstellungen](https://govhub.DOCS_BASE_DOMAIN/settings/profile) finden sich im Gov Hub in der Seitenleiste unter `Einstellungen > Profileinstellungen`.

## Kontoeinstellungen

Der Link "Kontoeinstellungen" öffnet sich in einem neuen Tab, der nach Änderung der gewünschten Einstellungen wieder geschlossen werden kann.

Hier können Vor- und Nachname geändert und die [Pseudonymisierung](./verwaltungsfunktionen.md#benutzer-pseudonymisierung) verwaltet und mit einem Klick auf "Speichern" bestätigt werden.
