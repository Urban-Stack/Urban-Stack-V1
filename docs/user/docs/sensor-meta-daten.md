# Sensor-Meta-Daten

Auf dieser Seite können Sensor-Meta-Daten als CSV-Dateien hochgeladen und verwaltet werden.

## Beispiel

Eine Beispieldatei mit fiktiven Daten kann [hier](/snippet/sensor-meta-data-example.csv) heruntergeladen werden.

## Hochladen
