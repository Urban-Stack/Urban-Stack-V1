# Mandanteneinstellungen

Die [Mandanteneinstellungen](https://govhub.DOCS_BASE_DOMAIN/settings/tenants) finden sich im Gov Hub in der Seitenleiste unter `Einstellungen > Mandanteneinstellungen`.

<figure>
  <img src="/screenshots/ugh-settings-tenant.webp" />
  <figcaption>Mandanteneinstellungen</figcaption>
</figure>

Hier können eine Reihe von mandantenspezifischen Eigenschaften geändert werden:

- Name des Mandanten: Der Anzeigename, der überall im Urbanstack verwendet wird, wenn nicht der technische Name genutzt werden muss.
- Logo: Link zu einem Bild, das in der oberen Menüleiste des Gov Hubs angezeigt wird.
- Titelbild Gov Hub: Link zu einem Bild, das auf der Startseite des Gov Hubs angezeigt wird.
- Titelbild Citizen Hub: Link zu einem Bild, das auf der Startseite des Citizen Hubs angezeigt wird.
- Farbe Gov Hub: Die Primärfarbe, die im Gov Hub verwendet wird.
- Farbe Citizen Hub: Die Primärfarbe, die im Citizen Hub verwendet wird.
- Neuigkeiten RSS Feed: Link zu einer Internetseite, die einen gültigen RSS-Feed bereitstellt.
- Koordinaten Rathaus: Die Koordinaten des Rathauses als Breiten- und Längenkoordinaten, getrennt von einem Doppelpunkt. Beispiel: `53.5438229:9.9889448`
- Link zum Impressum: Der Link zum Impressum, der im Citizen Hub angezeigt wird.
- Link zur Datenschutzerklärung: Der Link zur Datenschutzerklärung, der im Citizen Hub angezeigt wird.
- Link zu Kontaktdaten: Der Link zu Kontaktdaten, der im Citizen Hub angezeigt wird.

<!-- prettier-ignore -->
!!! warning "Bitte beachten!"
    Die Links zu Impressum, Datenschutzerklärung und Kontaktdaten **müssen** zwingend vorhanden sein, bevor der Citizen Hub des Mandanten genutzt werden kann!
